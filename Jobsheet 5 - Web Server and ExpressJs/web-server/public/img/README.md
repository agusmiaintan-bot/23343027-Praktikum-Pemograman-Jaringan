# INSTRUKSI: Tambahkan Foto Anda

Untuk menampilkan gambar di halaman **tentang.html**, silakan:

1. **Siapkan foto Anda** dalam format `.png` atau `.jpg`
2. **Copy foto** ke folder ini: `public/img/`
3. **Rename foto** menjadi `foto.png`

Atau, jika nama foto berbeda, edit file `tentang.html`:
```html
<img src="/img/NAMA_FOTO_ANDA.png">
```

Ganti `NAMA_FOTO_ANDA.png` dengan nama file foto Anda.

---

**Catatan**: Foto akan ditampilkan dengan lebar 250px (diatur di styles.css)
